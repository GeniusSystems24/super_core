# SuperCore Package Update Requirements

Update the Flutter package located at `flutter/super_core` to version `1.0.0`.

## Super Colors Palette

* Create a new class named `SuperPalette` that includes the following color palettes: `purplePalette`, `greenPalette`, `bluePalette`, `goldenPalette`, `grayPalette`, and `monochromePalette`.
* Each palette must contain 10 different color shades, ordered from the lightest to the darkest.
* Each palette must provide the colors required for primary, secondary, surface, background, border, text, disabled, error, success, and warning states.
* Ensure that `SuperPalette` follows the `GeniusLinkDesignSystem` requirements and provides all colors required to customize the application's overall appearance.

## Material Theme Data

* Create a new class named `SuperMaterialThemeData` with `light` and `dark` methods that generate `ThemeData` based on the `GeniusLinkDesignSystem`.
* The methods must accept a positional `SuperPalette` parameter using the following API:

```dart
SuperMaterialThemeData.light({SuperPalette palette = SuperPalette.bluePalette})
SuperMaterialThemeData.dark({SuperPalette palette = SuperPalette.bluePalette})
```

* The palette parameter may be optional, but its default value must be `SuperPalette.bluePalette`.
* Generate a complete `ColorScheme` from the provided `SuperPalette`.
* The generated `ColorScheme` must be different for light and dark modes while preserving the identity of the selected palette.
* All theme colors must be derived from the generated `ColorScheme` or the provided `SuperPalette`. Avoid unnecessary hardcoded colors.
* Enable Material 3 using `useMaterial3: true`.
* Configure the main Material theme properties, including typography, app bars, navigation components, buttons, form fields, cards, dialogs, tables, dividers, icons, and interactive states.
* Ensure appropriate color contrast and readability in both light and dark modes.
* Ensure that `SuperMaterialThemeData` follows the `GeniusLinkDesignSystem` requirements and includes all properties required to customize the application's overall appearance.
* Use the new theme system in all the following packages:

  * `flutter/super_tab_bar`
  * `flutter/super_auto_suggestion_box`
  * `flutter/super_form_field`
  * `flutter/super_map`
  * `flutter/super_navigation_page`
  * `flutter/super_navigation_sidebar`
  * `flutter/super_table_field`
  * `flutter/super_tree`
* Replace unnecessary hardcoded colors in these packages with values from `Theme.of(context).colorScheme` or the active `SuperPalette`.
* Ensure that all components automatically support both light and dark modes.
